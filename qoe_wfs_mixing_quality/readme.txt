Example Code to recreate the results reported in 
Deliverable D.6.2.3 Final Quality Model, 
Section 4.2 "Prediction of preference ratings for different music mixing techniques in 
context of spatial audio reproduction system"

This code was developed with Matlab R2015b, and requires in addition the Statistics and Machine Learning Toolbox.


How to use this example code:

1. Open fileProcessor.m 
2. Set flag to either compute features new or load them from prepared matlab file
3. Run fileProcessor.m to obtain results reported in Deliverable D.6.2.3
